Install Guide

Get Forge Download All Mods

Follow these steps (Windows):

Install Forge — download the installer from Forge Website , then run it and choose Install client.
Press Windows + R to open the Run dialog.
Type %appdata% and hit Enter.
Open .minecraft → mods folder. If mods doesn’t exist yet, launch the game once with Forge to generate it.
Put all your downloaded mods inside a folder named BA3O9-MC-Mods and place that folder inside .minecraft/mods.
Launch Minecraft with the Forge 1.21.10 – 60.0.18 profile, go to Multiplayer, add server 194.56.226.253:25662, join, and enjoy your stay.
